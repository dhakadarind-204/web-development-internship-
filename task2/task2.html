How validation works

Required fields
Name, Email, Subject, and Message must be filled in before the form submits.
Email format check
Validated using a regex: must include @ and a domain (e.g. user@mail.com).
Real-time feedback
Fields turn green ✓ when valid and red when something's wrong — as you type.
Phone (optional)
If entered, must be exactly 10 digits. If left blank, no error is shown.
Input types used
type="text"  type="email"  type="tel"  <select>  <textarea>  type="submit"
